# Repertoire - Landing Page

Landing page for **Repertoire**, a social media app for musicians combining practice tracking, audio sharing, and community.

the tech Stack:

- Vanilla HTML, CSS, and JavaScript
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts
- [Formspree](https://formspree.io) for waitlist form submissions

the Features:

- Fully responsive, mobile-first layout
- Interactive phone mockup with four tappable screens (Feed, Practice, Profile, Discover)
- Scroll-reveal animations
- Working waitlist form (submissions go to Formspree)
- No frameworks, no build tools, no `node_modules`
